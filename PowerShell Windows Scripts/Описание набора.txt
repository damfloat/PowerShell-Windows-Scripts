Версия набора 1.0

GUI - Набор скриптов, изменяющие интерфейс Windows 11

Network - Набор скриптов, взаимодействующих с сетью

# GUI:
- toggle_context_menu.ps1 - Возвращает старое контекстное меню, как в Windows 10, либо включает обратно новое

# Network:
- cf_dns.ps1              - Изменяет настройки DNS у всех адаптеров на Cloudflare + Google. Требуется перезагрузка после использования.
- list_open_ports.ps1     - Показывает все открытые TCP / UDP порты, статус, PID + Название процесса, что использует порт.
- reset_network.ps1       - Перезагрузка сетевых адаптеров + сброс DNS кеша + сброс TCP/IP стака + сброс Winsock.
